# 🎌 Anime Web Application

An experimental anime-focused web application built for learning, UI experimentation, and feature integration workflows.

## 🚀 Features

- Ongoing anime section
- Completed anime section
- Anime guessing functionality
- Interactive UI design experimentation
- AI chatbot integration experimentation

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- AI-assisted development workflows

## 📚 Purpose

This project was created to experiment with frontend development, anime-based content organization, and interactive web application concepts.

## 🚧 Status

The project is currently experimental and under active development.
